AngularJS1 "@uirouter" Target By Relative States Demo
=====================================================

可以使用`^.^`这样的记号来表示上级state，其中每个`^`就表示一个上级。

所以`^`表示parent state, `^.^`表示grandparent state.

```
npm install
npm run demo
```
